### APP Purpose
This app should provide instruments for creating characters for RPG game. The user should be able to CRUD characters, print the Charsheet for each character. He should also be able to create NPCs and monster characters, filter them out. There should be a separation between regular players and DM. THe app MIGHT provide a platform for having a game session for a group of users, so that they could do the skillchecks, interchange info and dynamically update the inventory, stats, e.t.c. The app's business logic should be flexible enough to provide for different RPG systems.
